# 博客系统（goserver）

博客系统是一个Go语言教学项目，遵循简洁架构，实现了Go项目开发过程中的绝大部分功能点，代码规范、质量高，并且可以作为企业应用的代码脚手架。

## Installation

安装步骤如下：

```bash
$ git clone https://github.com/marmotedu/goserver

$ cd goserver

# 安装依赖
$ make deps

# 安装依赖、编译二进制
$ make build
```
